Astra is a free WordPress theme available on the WordPress repository.The Astra Pro addon is a plugin that adds a range of powerful features that help you design better websites faster.

Yes, Astra and Starter Templates will work fine with all the standard hosting services that support WordPress.

Astra needs PHP version 5.3 and above. But we recommend you to use PHP version 7.3 or greater since it is a recommendation for WordPress.
